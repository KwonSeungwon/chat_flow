### Task 1: muteMember 응답 언래핑 수정

**Why:** 백엔드 `RoomMemberManagementController.muteMember`는 `ApiResponse.ok(new MuteResponse(...))`를 반환하므로 wire shape이 `{success, data: {mutedUntil}, ...}`이지만, 프론트는 root에서 `data['mutedUntil']`을 읽고 있어 **항상 null**. `DateTime.tryParse(null ?? **Why:** `MessageReadService`는 클래스 레벨 `@Transactional(readOnly = true)`. `getMessages`/`getMessagesByCursor`에서 호출하는 `decryptEntity`가 `entity.setContent(...)`로 영속 엔티티의 필드를 수정함. Hibernate dirty checking이 변경을 감지해 **메시지 페이지를 조회할 때마다 모든 엔티티에 UPDATE 쿼리가 실행**되며, 평문이 DB에 다시 쓰일 위험이 있다.

```java
    private ChatMessageEntity decryptEntity(ChatMessageEntity entity) {
        if (entity.getContent() == null) return entity;
        // IMPORTANT: never mutate the managed entity in a readOnly transaction —
        // Hibernate dirty checking would otherwise issue UPDATE statements and
        // could persist the plaintext back to the row. Build a detached copy.
        ChatMessageEntity copy = new ChatMessageEntity();
        copy.setId(entity.getId());
        copy.setChatRoomId(entity.getChatRoomId());
        copy.setUserId(entity.getUserId());
        copy.setUsername(entity.getUsername());
        copy.setContent(messageEncryptor.decrypt(entity.getContent()));
        copy.setType(entity.getType());
        copy.setTimestamp(entity.getTimestamp());
        copy.setMessageId(entity.getMessageId());
        copy.setPriority(entity.getPriority());
        copy.setAiGenerated(entity.isAiGenerated());
        copy.setFileUrl(entity.getFileUrl());
        copy.setFileName(entity.getFileName());
        copy.setFileContentType(entity.getFileContentType());
        copy.setParentMessageId(entity.getParentMessageId());
        copy.setParentMessagePreview(entity.getParentMessagePreview());
        copy.setDeleted(entity.isDeleted());
        copy.setEdited(entity.isEdited());
        copy.setEditedAt(entity.getEditedAt());
        copy.setPinned(entity.isPinned());
        copy.setReactions(entity.getReactions());
        copy.setForwardedFrom(entity.getForwardedFrom());
        return copy;
    }
```
